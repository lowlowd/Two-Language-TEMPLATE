## Qu'est-ce que le Don de Données Fizz?

OK, c'est l'une de ces fonctionnalités qui semble bizarre jusqu'à ce que vous l'utilisiez vraiment. Et ensuite, vous vous demandez pourquoi tous les opérateurs ne font pas ça.

**Le don de données** vous permet d'envoyer vos données mobiles inutilisées à d'autres utilisateurs Fizz. C'est tout. Pas de frais. Pas de piège. Vous avez 3 Go qui dorment et que vous n'utiliserez pas ce mois-ci? Envoyez-les à votre frère. Ou à un inconnu sur Reddit. Peu importe.

J'ai donné des données à des membres de ma famille qui semblent toujours être « à un appel vidéo du 0 $ ». Ça fonctionne parfaitement. Voici comment tout ça s'organise.

## Comment Fonctionne le Don de Données

Super simple :

1. Vous avez des données que vous n'utiliserez pas.
2. Vous les envoyez à un autre utilisateur Fizz.
3. Il les reçoit instantanément.
4. Ces données disparaissent de votre compte. Parties. Définitivement.

Pas de frais. Pas de piège. Juste du partage communautaire pur. Honnêtement, c'est plutôt bienveillant, non?

> **Vous voulez plus de données?** Utilisez le code de parrainage **HN9SM** lors de votre inscription pour obtenir un **bonus de 40 $**—de quoi vous payer beaucoup de données supplémentaires.

| ------------------------ | -------------------------- |
| Données du cycle actuel  | ✅ Oui                     |

## Étape par Étape : Comment Donner des Données sur Fizz

### Méthode 1 : Via l'Application ou le Site Web

1.  **Accédez à votre compte** sur [fizz.ca](https://fizz.ca) ou ouvrez l'application mobile.
2.  **Rendez-vous** dans l'onglet **Mon Forfait** (ou **Mes Services**).
3.  **Repérez** le bouton **Donner des données** (généralement identifié par une icône de cadeau).
4.  **Saisissez** le **numéro de téléphone** du destinataire (il doit s'agir d'un numéro Fizz actif).
5.  **Sélectionnez** la quantité de données à transférer (par blocs de Mo ou Go).
6.  **Validez** la transaction pour envoyer le cadeau.

Le destinataire reçoit les données immédiatement et votre solde s'ajuste en temps réel.

### Méthode 2 : Faire une demande de données

Vous avez besoin de données? Comme il n'existe pas de bouton « Demander » dans l'application Fizz, le processus est manuel :

1.  **Sollicitez un contact** directement (par message privé, texto ou via le forum communautaire).
2.  Laissez la personne effectuer le transfert vers votre numéro.
3.  **Surveillez votre compte** : les données apparaissent généralement en quelques minutes.

## Les Règles de Timing : Quand Pouvez-Vous Donner?

C'est là que les gens se trompent. Alors faites attention.

### Date Limite de Fin de Cycle

Vous pouvez donner des données jusqu'à **minuit le dernier jour de votre cycle de facturation**. Après ça? La fenêtre se ferme. Si vous essayez de donner des données après la fin de votre cycle, la transaction risque d'échouer.

Mon conseil : n'attendez pas jusqu'à 23h58 le jour de la date limite. Donnez un jour ou deux à l'avance. Mieux vaut prévenir que guérir.

### Timing du Report

- Vos données reportées les plus anciennes sont consommées en premier (principe du FIFO—premier entré, premier sorti).
- Après 2 mois, les données reportées non données... expirent simplement.
- Donc, si vous cumulez des données que vous n'utiliserez pas, donnez-les avant qu'elles ne disparaissent.

**Conseil pro :** Mettez un rappel de calendrier quelques jours avant la fin de votre cycle de facturation. Vérifiez votre solde. Donnez ce dont vous n'aurez pas besoin. Votre futur vous l'appréciera.

## Qu'arrive-t-il aux Données Données?

Quand vous donnez des données :

- Elles sont **définitivement retirées** de votre compte.
- Vous ne pouvez pas les récupérer.
- Si le destinataire ne les utilise pas, IL peut les reporter ou les redonner.

Quand vous recevez des données :

- Elles s'ajoutent à votre réserve de données actuelle.
- Elles comptent pour votre report si elles restent inutilisées.
- Il n'y a pas de pénalité d'expiration—elles sont traitées comme vos propres données.

## Puis-je Donner des Données Reportées?

**Oui!** C'est une question courante. Les données reportées peuvent être données exactement comme les données régulières. Fizz ne vous fait pas attendre et n'impose pas de restrictions spéciales.

Exemple :

- Vous avez un forfait de 5 Go.
- Vous avez utilisé 2 Go le mois dernier, donc 3 Go sont reportés.
- Ce mois-ci, vous avez 5 Go + 3 Go = 8 Go au total.
- Vous pouvez donner n'importe quelle portion de ces 8 Go.

## La Communauté d'Échange de Données

La fonctionnalité de don de Fizz a engendré une économie non officielle d'échanges de données :

### Où Trouver des Échanges de Données

1. **Reddit** - r/FizzMobile propose des fils de discussion réguliers pour les dons et échanges.
2. **Forum Fizz** - Le forum communautaire officiel a des discussions dédiées aux dons.
3. **Groupes Facebook** - Certains groupes d'utilisateurs Fizz facilitent les échanges.

### Comment Fonctionnent les Échanges

Types d'échanges typiques :

- **Dons à sens unique :** « J'ai 2 Go que je n'utiliserai pas, la première personne qui m'envoie un DM l'obtient. »
- **Échanges (Swaps) :** « Je te donne 1 Go maintenant, tu me donnes 1 Go le cycle prochain. »
- **Au suivant :** Quelqu'un vous donne des données, vous donnez à un autre inconnu le mois prochain.

### Est-ce Sécuritaire?

Quand vous donnez des données, vous partagez seulement votre numéro de téléphone Fizz. C'est tout. Vous NE donnez PAS :

- Vos informations de paiement
- Votre adresse
- Vos mots de passe de compte
- Quoi que ce soit d'autre de sensible

Mais voici la réalité honnête :

- **Le risque d'arnaque existe :** Quelqu'un promet de donner en retour le mois prochain, puis disparaît.
- **Asymétrie :** Vous donnez 2 Go en espérant 2 Go en retour... et plus de nouvelles.

Comment vous protéger :

- Commencez petit avec les inconnus (500 Mo, pas 5 Go).
- Restez avec des membres établis de Reddit ou du forum.
- Traitez les dons à sens unique comme exactement ça—un cadeau, pas un échange commercial.

## Considérations de Confidentialité

Quand vous donnez des données, le destinataire voit :

- Qu'il a reçu des données.
- Le montant (en Mo).
- Votre numéro de téléphone (l'expéditeur).

Le destinataire NE voit PAS votre :

- Nom (sauf si votre numéro a un afficheur associé).
- Détails du compte.
- Adresse ou informations de paiement.

Si vous n'êtes pas à l'aise de partager votre numéro de téléphone avec des inconnus, limitez vos dons à vos amis et votre famille.

## Et les Forfaits Illimités?

Si vous êtes sur un forfait illimité Fizz :

- Vous n'avez pas de solde de données à donner (puisque c'est illimité).
- Vous POUVEZ recevoir des dons des autres (bien que l'utilité soit limitée).
- Votre plafond de données haute vitesse ne se réinitialise pas avec les dons reçus.

Les utilisateurs de forfaits illimités bénéficient moins du don de données, mais peuvent quand même techniquement en recevoir.

## FAQ : Don de Données Fizz

### Pourquoi je ne peux pas donner de données à un ami spécifique?

Il y a généralement deux raisons :

**Pas de Forfait Données :** Le destinataire doit avoir des données mobiles dans son forfait mensuel. Vous ne pouvez pas donner des données à un abonné qui a seulement un forfait « Voix et Textos ».

**Mode Incognito :** Si votre ami a activé le mode « Incognito » dans ses paramètres de confidentialité, il ne peut pas être trouvé dans l'outil de don. Demandez-lui de passer temporairement à « Expérience Fizz complète ».

### Y a-t-il une limite minimum ou maximum de don de données?

Oui. Vous devez donner par blocs de 500 Mo. Vous ne pouvez pas envoyer des montants personnalisés comme 100 Mo. Il n'y a pas de limite maximum—vous pouvez donner tout le solde de votre cycle actuel si vous le souhaitez, tant que ce soit fait par tranches de 500 Mo.

### Le destinataire voit-il mon numéro de téléphone quand je donne?

Oui. Quand vous donnez des données, le destinataire verra votre Prénom et votre Numéro de Téléphone. Il ne verra pas votre courriel, votre adresse ou les détails de votre compte. Gardez cela en tête si vous donnez à des inconnus sur les forums communautaires.

### Puis-je transférer des données entre forfaits sur mon propre compte?

Oui. Si vous gérez plusieurs forfaits (ex. le vôtre et celui d'un enfant) sous une seule connexion, vous pouvez facilement déplacer des données entre eux. Allez sur le forfait ayant des données en trop, cliquez sur **Donner**, et sélectionnez l'autre forfait de votre compte comme destinataire. Le transfert est instantané.

### Puis-je donner des données à plusieurs personnes?

Oui! Donnez autant de fois que vous le souhaitez, tant que vous avez des données disponibles.

### Le don coûte-t-il quelque chose?

Non. Le don est entièrement gratuit. Il n'y a aucun frais pour envoyer ou recevoir.

### Et si je donne des données par erreur?

Il n'y a pas d'annulation possible. Les données données sont définitivement retirées de votre compte. Vérifiez bien avant de confirmer.

### Les données données comptent-elles pour Mes Récompenses?

Non. Donner des données ne vous fait pas gagner de points ou de niveaux. Seuls la facturation, les parrainages et la participation communautaire permettent de gagner des récompenses.

### Puis-je donner mes données restantes juste avant d'annuler Fizz?

Techniquement oui, mais vérifiez le timing. Les dons de données sont traités dans votre cycle de facturation, donc si vous annulez en milieu de cycle, il peut y avoir des complications.
