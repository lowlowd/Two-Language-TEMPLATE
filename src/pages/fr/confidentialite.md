**Dernière mise à jour : décembre 2025**

## Introduction

Code promo Fizz (« nous », « notre » ou « nos ») exploite le site web fizzpromocode.ca. Cette politique de confidentialité explique comment nous recueillons, utilisons et protégeons vos informations lorsque vous visitez notre site web.

## Informations que nous recueillons

### Informations que vous fournissez

- Coordonnées lorsque vous utilisez notre formulaire de contact
- Adresse courriel si vous vous abonnez aux mises à jour

### Informations recueillies automatiquement

- Type et version du navigateur
- Pages visitées et temps passé
- Site web référent
- Adresse IP (anonymisée)

## Comment nous utilisons vos informations

Nous utilisons les informations que nous recueillons pour :

- Répondre à vos demandes
- Améliorer notre site web et notre contenu
- Analyser le trafic et les tendances d'utilisation du site
- Envoyer des communications promotionnelles (avec votre consentement)

## Services tiers

Nous utilisons les services tiers suivants :

- **Google Analytics** - Pour l'analyse du site web
- **Vercel** - Pour l'hébergement du site web

Ces services peuvent recueillir et traiter des données selon leurs propres politiques de confidentialité.

## Divulgation d'affiliation

Ce site web contient des liens d'affiliation vers Fizz. Lorsque vous vous inscrivez en utilisant notre code promo, nous pouvons recevoir un bonus de parrainage. Cela n'affecte pas le prix que vous payez.

## Témoins (cookies)

Nous utilisons des témoins pour analyser le trafic du site web et améliorer l'expérience utilisateur. Vous pouvez désactiver les témoins dans les paramètres de votre navigateur.

## Vos droits

Vous avez le droit de :

- Accéder à vos données personnelles
- Demander la suppression de vos données
- Vous désabonner des communications promotionnelles
- Demander la portabilité des données

## Sécurité des données

Nous mettons en œuvre des mesures de sécurité appropriées pour protéger vos informations personnelles. Cependant, aucune transmission internet n'est complètement sécurisée.

## Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité de temps en temps. Nous vous informerons de tout changement en publiant la nouvelle politique sur cette page.

## Nous contacter

Si vous avez des questions concernant cette politique de confidentialité, veuillez nous contacter via notre page de contact.
